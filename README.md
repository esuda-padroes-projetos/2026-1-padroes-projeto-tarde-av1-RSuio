# 2026-1-padroes-projeto-tarde-av1-RSuio
Titulo - Ferro Velho Fit (FVFit)

Integrantes: Alberto Vitorino, Bruno De Melo, Caique Nunes.

Descrição: Vamos fazer um interface de cadastro de alunos para uma academia de musculação, com um sistema novo de planos, anuais, mensais, semestrais, diaria. Contratação de profissionais da área da saude como: Personal, nutricionista e fisioterapeuta. Uma interface para mostrar uma breve descrição da academia. Espaços livres para musculação, dança, cardio.
